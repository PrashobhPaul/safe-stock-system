How StockSage Scores Stocks
Each stock is scored 0–100 using a five-category weighted rule-based engine processing 20+ technical indicators. No machine learning, no LLMs — pure quantitative rules applied consistently across Nifty 100 + Midcap 150 universe.
Trend Analysis
30 / 100 pts
  • SMA 20/50/200 alignment (bullish stack)
  • Price vs SMA200 distance (optimal zone)
  • EMA 12/26 crossover detection
  • ADX trend strength (>25 = strong)
  • Higher highs / higher lows structure
Momentum
25 / 100 pts
  • RSI zone scoring (50–65 = optimal)
  • MACD line vs signal line crossover
  • MACD histogram direction & magnitude
  • Stochastic Oscillator (%K/%D)
  • Rate of Change (ROC) 10-day
Volume
20 / 100 pts
  • Volume vs 20-day average ratio
  • OBV (On-Balance Volume) trend slope
  • Volume-Price confirmation (3-day)
  • Accumulation/Distribution indicator
  • Volume surge detection (>2x spike)
Breakout
15 / 100 pts
  • Bollinger Band position (% within band)
  • Bollinger Band squeeze detection
  • 20-day high breakout confirmation
  • Price channel analysis
  • ATR-based volatility regime
Price Action
10 / 100 pts
  • 52-week high proximity (< 10% = bullish)
  • 5-day return momentum
  • Today's candle direction (close vs open)
  • Support/resistance level proximity
  • Gap-up / gap-down analysis
Signal Thresholds
● STRONG BUY ≥ 75 ● BUY 60–74 ● WATCH 45–59 ● AVOID < 45
Stock Universe
Nifty 50 + Nifty Next 50 + Nifty Midcap 100 (≈ 200 stocks analyzed daily)
Data Source
Yahoo Finance API (yfinance) · NSE India · 1 year of daily OHLCV
